# Personal Finance Dashboard

An end-to-end Personal Finance Dashboard built in Power BI to track income, expenses, and savings — answering the core question: *Where is my money going, and am I on track financially?*

## Tech Stack
`Power BI` `DAX` `Power Query` `Excel` `Star Schema` `ETL`

## Key Insights
- 22% savings rate maintained consistently over 3 years (2018–2021)
- House Rent alone consumed 40% of total expenses — biggest financial pressure point
- 71% of savings directed into Mutual Funds — investment-first approach
- Net worth of ₹3.25 Lakh generated over the period
